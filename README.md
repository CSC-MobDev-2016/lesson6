# File manager
Напишите файловый менеджер.

## Base features
 - Список папок и файлов с навигацией вверх и вниз
 - Открытие файлов (`Uri.fromFile()` + `ACTION_VIEW`)
 - Разные иконки для файлов и папок
 - RecyclerView 
 - Контекстное меню для операций с файлами: открыть/удалить/переименовать (используйте `AlertDialog`)  

## Extra points
 - Системные иконки для файлов + превью картинок
 - Собственный просмотрщик картинок
 - Групповые операции

## Penalty
 - Все предыдущие
